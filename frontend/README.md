# Smart Mentor Frontend

React + TypeScript + Tailwind CSS interface for the Smart Interview Preparation Mentor.

## Run

```bash
npm install
npm run dev
```

The Vite dev server proxies `/api` to `http://localhost:8000`.

## Pages

- `/` landing page
- `/login`
- `/dashboard`
- `/resume`
- `/interview`
- `/coding`
- `/analytics`
- `/settings`
